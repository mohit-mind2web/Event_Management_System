# CodeIgniter 4 Application Starter

## What is CodeIgniter?
CodeIgniter is a PHP full-stack web framework that is light, fast, flexible and secure.
More information can be found at the [official site](https://codeigniter.com).

## Event Management System
This project is built using CodeIgniter 4.
It is an Event Management System for organizers and users to create, manage, and register for events.

## Installation & Setup
1. Copy `env` to `.env` and configure baseURL & database.
2. Use `composer install` to install dependencies.

## Server Requirements
PHP 8.1+ with `intl` and `mbstring` extensions, and other extensions as mentioned above.
